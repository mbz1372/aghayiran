# آقای ایران - فروشگاه دوچرخه و لوازم جانبی

پروژه آماده deploy روی Vercel با Next.js 14، TypeScript، Tailwind و طراحی فارسی RTL.

## اجرا
```bash
npm install
npm run dev
```

## بیلد
```bash
npm run build
```

## دیپلوی روی Vercel
1. پروژه را در GitHub آپلود کنید.
2. در Vercel گزینه New Project را بزنید.
3. ریپو را انتخاب کنید.
4. Framework: Next.js
5. Build Command: `npm run build`
6. Output: پیش‌فرض Next.js
7. Deploy

## صفحات
- `/` صفحه اصلی
- `/shop` فروشگاه
- `/product/p1` صفحه محصول
- `/cart` سبد خرید
- `/account` ورود
- `/categories` دسته‌بندی
- `/blog` راهنمای خرید
- `/about` درباره ما
- `/contact` تماس
- `/admin` پنل مدیریت نمایشی
