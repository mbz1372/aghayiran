# آقای ایران - نسخه موبایل‌فرندلی فروشگاه دوچرخه

نسخه Mobile-first و آماده Deploy روی Vercel با Next.js 14، TypeScript و Tailwind.

## دیتای محصول
دسته‌بندی و مدل دیتا با الهام از فروشگاه‌های واقعی دوچرخه مثل Decathlon ساخته شده:
- Mountain bikes
- Road bikes
- Gravel bikes
- City bikes
- Kids bikes
- Helmets
- Lights
- Locks
- Pumps & tools
- Bags & bottles

نام‌ها، متن‌ها و قیمت‌ها اختصاصی و نمونه هستند و کپی مستقیم از سایت اصلی نیستند.

## ویژگی‌های موبایل
- Bottom Navigation موبایل
- Search تمام‌صفحه موبایل
- کارت محصولات سبک‌تر
- فیلتر چیپ افقی
- Sticky add-to-cart در صفحه محصول موبایل
- سبد خرید و چک‌اوت مناسب موبایل
- صفحات فروشگاه، محصول، دسته‌بندی، جستجو، سبد، چک‌اوت، حساب، بلاگ، تماس، ادمین

## اجرا
```bash
npm install
npm run dev
```

## Build
```bash
npm run build
```
