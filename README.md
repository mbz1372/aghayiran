# آقای ایران - نسخه حرفه‌ای فروشگاه دوچرخه

یک پروژه کامل و آماده Vercel با Next.js 14، TypeScript، Tailwind، RTL و طراحی فارسی.

## ویژگی‌ها
- صفحه اصلی حرفه‌ای و لاکچری
- فروشگاه با فیلتر نمایشی
- صفحه محصول کامل با گالری، ویژگی‌ها، گارانتی، ارسال و محصولات مشابه
- سبد خرید و چک‌اوت
- دسته‌بندی، مقایسه محصولات، مجله، گارانتی، تماس، درباره ما
- پنل مدیریت نمایشی
- کاملاً قابل Deploy روی Vercel

## اجرا
```bash
npm install
npm run dev
```

## Build
```bash
npm run build
```

## Deploy روی Vercel
1. فایل‌ها را داخل GitHub بگذارید.
2. در Vercel گزینه New Project را بزنید.
3. ریپو را انتخاب کنید.
4. Framework: Next.js
5. Build Command: `npm run build`
6. Deploy

## نکته
این نسخه Frontend کامل است. برای پرداخت واقعی، دیتابیس، انبارداری و احراز هویت واقعی باید به API/Database وصل شود.
