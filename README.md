# آقای ایران - فروشگاه دوچرخه + پنل مدیریت فروش، فاکتور و انبار

نسخه کامل‌تر و آماده Deploy روی Vercel با Next.js 14، TypeScript و Tailwind.

## امکانات فروشگاه
- صفحه اصلی Mobile-first
- فروشگاه، دسته‌بندی، جستجو، محصول، سبد خرید، چک‌اوت
- دیتای نمونه محصول، قیمت، قیمت تمام‌شده، موجودی، فروش و گارانتی

## امکانات پنل مدیریت
مسیر اصلی پنل:
`/admin`

صفحات پنل:
- `/admin` داشبورد مدیریتی
- `/admin/orders` مدیریت سفارش‌ها
- `/admin/invoices` مدیریت فاکتورها
- `/admin/inventory` مدیریت موجودی انبار
- `/admin/products` مدیریت محصولات
- `/admin/customers` مدیریت مشتریان
- `/admin/reports` گزارش‌های فروش و سود

## بخش‌های پنل
- KPI فروش، سود، ارزش موجودی خرید و فروش
- لیست سفارش‌ها با وضعیت پرداخت/ارسال/لغو
- لیست فاکتورها با وضعیت پرداخت‌شده، پرداخت‌نشده و سررسید گذشته
- ثبت نمایشی گردش انبار: ورود، خروج، اصلاح
- هشدار کمبود موجودی بر اساس حداقل موجودی
- مدیریت محصولات با قیمت خرید، فروش، SKU، موجودی
- گزارش شهرها و کالاهای پرفروش

## اجرا
```bash
npm install
npm run dev
```

## Build
```bash
npm run build
```

## Deploy روی Vercel
1. فایل‌ها را در GitHub آپلود کنید.
2. در Vercel گزینه New Project را بزنید.
3. ریپو را انتخاب کنید.
4. Framework: Next.js
5. Build Command: `npm run build`
6. Deploy

## نکته مهم
این نسخه Frontend و دیتای نمونه دارد. برای استفاده واقعی باید به دیتابیس مثل PostgreSQL/Supabase و API پرداخت وصل شود.
